<h2 align="center" id="title">E-Badge <br> PDF Ticket Generator with Python (Anvil)</h2>

<p id="description">Web application developed with Python using Anvil, allowing users to register for an event and automatically generate an electronic ticket (E-Badge) in PDF format, available for download and sent by email.</p>

<h2>Project Goal</h2>

*   Generate dynamic PDF tickets from an Anvil form
*   Validate user inputs before submission
*   Automatically download the generated ticket
*   Send the ticket by email to the participant


<h2>Main Features</h2>

*   Registration form (nom, fonction, organisation, mail)
*   Field validation
*   Automatic PDF generation
*   Ticket download
*   Email sending
*   Custom PDF file naming

<h2>Clone & Run the Project with Anvil :</h2>

Official Method (Recommended)

<p>1. Log in to your Anvil account :</p>

```
https://anvil.works/
```

<p>2. From the Dashboard :</p>

```
Click on Create a new new app
```

<p>3. Select :</p>

```
Clone from GitHub
```

<p>4. Paste the GitHub repository URL :</p>

```
https://github.com/adjagbafortune/e-badge_projet.git
```

<p>5. Click Clone App :</p>

The project is immediately ready to run and modify.


<h2>Run the App</h2>

*   Click Run
*   Fill in the form
*   Click "obtenir mon badge"
*   The PDF ticket will be :
    *   downloaded
    *   and sent by email
